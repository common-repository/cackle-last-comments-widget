== Description ==

This plugin integrates "Cackle Last Comments Widget" as sidebar widget into your website.

== Installation ==

1. Upload  folder `cackle-last-comments-widget` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to wp-admin/widgets.php
4. Move `cackle-last-comments-widget` to sidebar
5. Enter params(mcSite, mcSize, mcAvatarSize, mcTextSize) from cackle.ru
